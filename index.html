Bienvenue sur le site !

Sur ce site en partenariat avec la fondation Race for Water, vous allez pouvoir interagir avec une simulation qui représente les parallèles entre les systèmes humains et les systèmes océaniques. Vous pourrez explorer comment chaque partie du corps humain trouve un écho dans les mécanismes de l'océan. Cette simulation illustre les bénéfices d'un bon fonctionnement comme les conséquences des dysfonctionnements.
